Act as a creative frontend developer and game UI designer.

Create a highly interactive **gaming-themed personal portfolio website** using **HTML, CSS, and JavaScript only (no frameworks)**. The design should feel like a **video game interface (HUD/UI)** with neon effects, futuristic elements, and immersive animations.

Design Theme:

* Dark cyberpunk / gaming aesthetic
* Neon glow colors (blue, purple, green)
* Glassmorphism + glowing borders
* Game-style UI panels and cards
* Animated background (particles, grid, or stars)
* Sound effects on clicks (optional but recommended)
* Smooth transitions and micro-interactions

Layout Style:

* Looks like a **game dashboard / control panel**
* Sections appear like “levels” or “missions”
* Cursor effects (glow or trail)
* Loading screen like a game intro

Website Sections (Game-Based Naming):

1. **Start Screen (Hero Section)**

   * Full-screen intro like a game start menu
   * Text: "Press Start" or "Enter Portfolio"
   * Animated typing effect
   * Player Name (Your Name)
   * Role: "Software Developer / AI Engineer"
   * Buttons: Start Game / View Portfolio / Download Resume

2. **Player Profile (About Me)**

   * Avatar/profile image
   * Stats-style layout:

     * Level (Education)
     * Experience Points
     * Skills unlocked
   * Short bio

3. **Skill Tree (Skills & Technologies)**

   * Skills displayed like a **game skill tree**
   * Categories:

     * Languages
     * Frameworks
     * Tools
   * Animated progress bars or unlock effects

4. **Missions (Projects Section)**

   * Projects shown as missions/cards
   * Each mission includes:

     * Title
     * Description
     * Tech stack
     * GitHub link
     * Live demo
   * Hover effect: glow + expand

   Example missions:

   * Spam Detection AI
   * Social Network Graph System
   * Speed Racer Game
   * AI Dashboard

5. **Achievements (Certifications & Courses)**

   * Display as badges or trophies
   * Hover animations

6. **Experience Log (Internships / Work)**

   * Timeline like game progress
   * Show roles and achievements

7. **Leaderboard (Competitive Programming)**

   * Coding stats
   * Platforms like LeetCode, CodeChef

8. **Open Source Arena**

   * GitHub contributions
   * Repository cards

9. **Game Journal (Blog / Optional)**

10. **Secret Lab (Research / Patent)**

11. **Contact Terminal**

* Futuristic contact form
* Fields: Name, Email, Message
* Neon glowing input fields

12. **Exit / Resume Download**

* Download Resume button
* Styled as "Download Player Data"

Technical Requirements:

* Pure HTML, CSS, JavaScript
* CSS animations + keyframes
* JavaScript for:

  * Typing effect
  * Scroll animations
  * Interactive UI
* Fully responsive design
* Organized file structure (index.html, style.css, script.js)

Advanced Features:

* Custom cursor effect (glow trail)
* Background animation (particles or grid)
* Sound effects on button click
* Smooth scroll and section reveal
* Game-style loading screen
* Hover glow effects
* Scroll progress indicator

Deliver:

* Full working code (HTML, CSS, JS)
* Clean and well-commented code
* Instructions to run locally

Goal:
The website should feel like an **interactive game experience**, not just a normal portfolio. It should impress recruiters and trainers within 2 minutes of presentation.
